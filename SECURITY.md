## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in this project, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

Please create a report and send it via email to <mark>[INSERT CONTACT METHOD]</mark>.

Please include:
* A description of the vulnerability.
* Steps to reproduce the issue.
* Any relevant logs or screenshots.

### Response Timeline

We will verify the receipt of your report within **48 hours** and will provide you with an estimated timeline for the assessment and fix.

If the vulnerability is accepted, we will credit you in our release notes (unless you wish to remain anonymous).
